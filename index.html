Introduction a la programmation web

  1. Serveurs web


    2. Un site est constitué matériellement,d'un ordinateur connecté à l'internet,et d'un programme tournant en permanence sur cet ordinateur,le serveur.


  2. Client web

    3. Le client est couramment appelé un navigateur.Les navigateurs les plus connus étant Netscape,Internet Explorer....


    Les navigateurs sont des logiciels soignesement étudiés pour faciliter et assister la navigation sur le web.


    Ils lisent et interprettent des languages HTML de manière a donner un affichage des pages web.
